# 台糖工具包

台糖工具包提供許多實用功能，自動化工作中的常規和繁瑣任務，藉此提高效率並減少處理重複性的手動工作。

![Screenshot](images/purchase-order-screenshot.png)

## 功能

- **報表生成**：自動生成各種贈品（如面紙和瓶裝水）的每週訂貨通知單。
